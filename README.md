# timeWorkApp
time work app for RAOT develop by ionic2

#### จำเป็นต้องใช้ 
- typings: 1.0 ขึ้นไป
- ionic: 2.0 ขึ้นไป
- cordova

## ก่อนใช้งาน
```
npm install -g ionic@beta typings cordova ts-node
```

## การติดตั้ง
```
npm install
```

## เวลาพัฒนา
```
npm run dev
```

## Run
### ionic
```
ionic serve
```
### Server
```
npm run server
```
